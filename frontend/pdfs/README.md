# Resume PDF Files

Place your 4 resume PDFs in this folder:

1. **modern_resume.pdf** - Modern template PDF
2. **classic_resume.pdf** - Classic template PDF  
3. **creative_resume.pdf** - Creative template PDF
4. **minimal_resume.pdf** - Minimal template PDF

These PDFs will be displayed on the left side of the preview page alongside the Overleaf code.

## How to generate PDFs from Overleaf:

1. Create a project in Overleaf with the LaTeX code
2. Click "Download PDF" button in Overleaf
3. Save it as `[template-name]_resume.pdf`
4. Place it in this pdfs folder

The preview page will automatically load these PDFs when you switch between templates.
